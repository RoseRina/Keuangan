# Aplikasi Manajemen Keuangan

Aplikasi web sederhana untuk mencatat pemasukan dan pengeluaran bulanan, seperti pembayaran PDAM, Indihome, dan pengeluaran lainnya.

## Fitur

- Pencatatan pemasukan dan pengeluaran
- Kategori transaksi (PDAM, Indihome, dll)
- Ringkasan keuangan (total pemasukan, pengeluaran, dan saldo)
- Riwayat transaksi
- Format mata uang Rupiah
- Tampilan responsif

## Cara Menjalankan

1. Install dependencies:
   ```bash
   npm install
   ```

2. Jalankan aplikasi:
   ```bash
   npm start
   ```

3. Buka browser dan akses `http://localhost:3000`

## Teknologi yang Digunakan

- React.js
- Tailwind CSS
- Heroicons
